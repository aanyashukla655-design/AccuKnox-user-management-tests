# AccuKnox User Management Tests

This project includes the User Management E2E flow for OrangeHRM.  
I tried to keep the tests simple and readable. I followed Playwright with Page Object Model.

## What is covered
- Login
- Navigate to Admin module
- Add a new user
- Search user
- Edit user details
- Validate after edit
- Delete user
- Validate delete
